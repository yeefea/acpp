## 内存管理

3种内存管理
- automatic自动，局部变量离开作用域自动释放内存（禁止返回局部变量的指针）
- static，可以返回静态变量的指针
- dynamic，可以返回new创建的指针，需要手工delete或delete[]

### dynamic allocating
new会初始化内存
```c++

T *p = new T;
T *p = new T(args);
// 对应
delete p;

T *p = new T[n];
// 对应
delete[] p;
```
